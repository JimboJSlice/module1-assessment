# Kobe McDonald Website Project

## Description

This project created a clean, modern and responsive website to highlight the GAA Mayo footballer that is Kobe McDonald. The website created is a place where fans can discover all things Kobe.
Latest imagery, videos, stats, news and even a way to contact him. This website has it all.

## Features

- Semantic HTML
- A mobile-first responsive layout
- Responsive Nagation
- Embedded YouTube Video
- Table And Form
- CSS Transition
- Bootstrap Gallery
- Accessibility best practices
- Custom styling with well structured code

## Accessibility Notes

The website was tested using Microsoft's Narrator screen reader to ensure content could be navigated effectively. Semantic HTML elements were used to improve compatibility with assistive technologies. All images included have meaningful alternative text. Colour choices were selected to provide sufficient contrast. Responsive testing was carried out across different screen sizes and modern web browsers. The website was also validated using the W3C HTML and CSS validation to identify and correct coding issues before completion.
